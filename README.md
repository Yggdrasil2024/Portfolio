# Portefolio

portefolio personnelle formé en 7 sections. concut dans le cadre de la formation d'Akieni academy. il s'agit du fruit d'un travail laborieux mais necessaire ainsi que l'aboutissement de plus d'un mois d'apprentissage en aspirant à gagner encore plus en competences à l'avenir.

## fonctionnalités

- html semantique: structure des pages html pensée sans surchage de balise div
- menu humberger: menu humberger sans js en pure css
- responsivité mobile et desktop
- formulaire de contacts

## demonstration

[demonstration en live du site](https://yggdrasil2024.github.io/Portfolio/)

## Technologies utilisé

1. html 5: structuration
2. css 3: feuille de style
3. google fonts: pour l'integration des fonts `Poppins`
4. Git: pour le versionning

## Structure du projet

```txt
├── index.html          # Page d'accueil (Hero, Présentation, Valeurs)
├── style.css           # Feuille de style globale (Mobile & Desktop)
└── assets/
    └── icons/          # Icônes et ressources graphiques SVGs
    └── img/            # dossiers contenant les imgaes
```

## Obtenir le projet

N'hesiter surtout pas à recuperer le projet pour pouvoir le vous en inspirer ou y mettre un peu de votre creativité pour l'ameliorer;

1. Cloner le repo

```bash
git clone https://github.com/Yggdrasil2024/Portfolio.git
```

2. acceder au dossier

```bash
cd Portfolio
```
